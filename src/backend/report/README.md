# Report index

## Purpose
This folder stores reports and notes about changes, testing, and next steps.

## Structure
- changes/ : change logs per date
- testing/ : testing guides and checks
- next-steps/ : missing items, risks, and planned work

## Naming
- changes/YYYY-MM-DD.md
- testing/how-to-test.md
- next-steps/missing-and-next.md
